HTTP protokolünü kullanarak kaynaklara erişimi sağlayan bir API tasarım yaklaşımıdır. Bu API, kaynakları temsil eden URI'ları (Uniform Resource Identifier) kullanır ve HTTP metotlarını (GET, POST, PUT, DELETE) kullanarak kaynaklar üzerinde işlemler gerçekleştirir.
